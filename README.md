
# Simple Engine

A simple application engine written in C.
Handles windowing, rendering, and reading
inputs.


Currently It supports only X11 and uses 
Vulkan graphics API. The plan is to eventually
support Wayland and Windows as well. Mac support
is not planned because Mac is insanely cursed
and requires Objective C. Maybe one day I'll
set that up with a custom build script
but that'll be far far in the future if ever.
